# SkillForge – Career Management Dashboard

SkillForge is a responsive career management dashboard built using HTML, CSS, and JavaScript. It helps users manage job applications, track interviews, monitor skill development, and organize certifications in a single platform.

## Features

### Dashboard

* Career analytics dashboard
* Application statistics
* Status-based insights
* Interactive charts using Chart.js

### Job Tracker

* Add job applications
* Edit existing records
* Delete applications
* Search jobs by company
* Filter jobs by status
* Local Storage support

### Skills Tracker

* Add technical skills
* Track skill proficiency percentage
* Visual progress bars
* Delete skills
* Persistent data storage

### Interview Tracker

* Track interview schedules
* Manage interview rounds
* Monitor interview status
* Delete interview records
* Local Storage integration

### Certificate Tracker

* Store certification details
* Add certificate provider information
* Save completion dates
* Add certificate links
* Delete certificates

### Additional Features

* Dark Mode support
* Responsive design
* Multi-page navigation
* Font Awesome icons
* Local Storage persistence
* Modern UI/UX

## Tech Stack

* HTML5

* JavaScript (ES6)
* Chart.js
* Local Storage API
